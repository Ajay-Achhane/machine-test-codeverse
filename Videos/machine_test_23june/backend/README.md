# 🚀 Ajay - Node.js REST API Machine Test

A scalable and production-ready Node.js REST API boilerplate using **Express, MongoDB, and Mongoose ORM**.

---

## 📦 Features

* 🔐 Authentication using JWT
* 🔑 Password hashing with bcrypt
* 📦 Mongoose ORM (MongoDB)
* 📁 MVC Folder Structure
* 🛡️ Validation using validatorjs
* 🌐 CORS enabled
* ⚙️ Environment-based config (.env)

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* Mongodb
* Mongoose ORM

---


## 🚀 Usage

1. Clone or install package
2. Setup `.env` file

```env
PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=yourdbname
JWT_SECRET=your_secret
```

3. Run project

```bash
npm run dev
```
